# postgres-jalali
Docker image of latest postgresdb with jalali utils installed

# Built images

```bash
docker pull khodaparastan/postgres-jalali:latest
```
# Architectures supported

- linux/amd64

- linux/arm64

- linux/arm/v7

- linux/arm/v6
